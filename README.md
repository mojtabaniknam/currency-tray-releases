# WinArz — نرخ ارز لحظه‌ای برای ویندوز

نرم‌افزار رایگان نمایش لحظه‌ای نرخ ارز، طلا و سکه در نوار وظیفه ویندوز.

---

## دانلود

آخرین نسخه را از بخش [Releases](https://github.com/mojtabaniknam/WinArz-releases/releases) دریافت کنید.
فقط فایل `WinArz.exe` را دانلود و اجرا کنید — نیازی به نصب نیست.

---

## راهنمای شروع

### ۱. دریافت کلید API
برای دریافت نرخ ارز، نیاز به کلید رایگان از یکی از سرویس‌های زیر دارید:

| سرویس | لینک دریافت کلید | توضیح |
|-------|-----------------|-------|
| **نوسان** (پیشنهادی) | [navasan.tech/api](https://www.navasan.tech/api/) | رایگان |
| **BRS API** | [brsapi.ir](https://brsapi.ir/free-api-gold-currency-webservice/) | رایگان |
| **آلان‌چند** | [alanchand.com/media/api](https://alanchand.com/media/api) | محدودیت رایگان |

### ۲. اجرا و تنظیم
1. فایل `WinArz.exe` را اجرا کنید
2. روی دکمه **⚙** (تنظیمات) در پنجره اصلی کلیک کنید
3. از تب **API** سرویس مورد نظر را انتخاب کرده، کلید API را وارد کنید
4. از تب **ارزها** ارزهای مورد نظر را انتخاب کنید
5. روی **ذخیره** کلیک کنید — نرخ‌ها نمایش داده می‌شوند

---

## ماشین حساب تبدیل ارز

در پنجره اصلی، کنار دکمه تنظیمات دکمه **⊞** وجود دارد:
- کلیک روی آن ماشین حساب را باز می‌کند
- **مبلغ** را وارد کنید
- **ارز** مبدا/مقصد را انتخاب کنید
- نتیجه به‌صورت خودکار محاسبه می‌شود
- دکمه **⇄** برای جابجایی مبدا و مقصد

## امکانات

- نمایش نرخ ارز، طلا و سکه در نوار باریک پایین صفحه
- نوسانات لحظه‌ای با رنگ‌بندی سبز/قرمز
- ماشین حساب تبدیل تومان ↔ ارز
- اعلان ویندوز هنگام آپدیت یا خطا
- آپدیت خودکار نرم‌افزار
- اجرا هنگام ورود به ویندوز
- پشتیبانی از ۳ سرویس API (نوسان، BRS API، آلان‌چند)

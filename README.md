# WinArz — نرخ ارز لحظه‌ای برای ویندوز

نوار کوچکی که کنار ساعت ویندوز قرار می‌گیرد و نرخ ارز، سکه، طلا و رمزارز را به‌صورت لحظه‌ای نمایش می‌دهد.

---

## دانلود

⏬ آخرین نسخه را از صفحه [Releases](https://github.com/mojtabaniknam/WinArz-releases/releases) دانلود کنید.  
فایل `WinArz.exe` را اجرا کنید — نصب نیاز ندارد.

---

## ⚠ هشدار Windows SmartScreen

هنگام اولین اجرا، ممکن است ویندوز پیام آبی‌رنگ **"Windows protected your PC"** نمایش دهد.  
این هشدار برای **همه** برنامه‌های بدون امضای دیجیتال (Code Signing) طبیعی است.

**برای رد کردن هشدار:**

> ۱. روی **More info** کلیک کنید  
> ۲. دکمه **Run anyway** ظاهر می‌شود — روی آن کلیک کنید  
> ۳. برنامه اجرا می‌شود

این هشدار فقط در اولین اجرا نمایش داده می‌شود.

---

## معرفی برنامه

WinArz یک ابزار سبک و همیشه در دسترس است که بدون نیاز به باز کردن مرورگر یا اپلیکیشن جداگانه، نرخ لحظه‌ای ارز، طلا، سکه و رمزارز را روی صفحه شما نگه می‌دارد.

برنامه از دو بخش اصلی تشکیل شده است:

| بخش | توضیح |
|-----|-------|
| **نوار اسلایدینگ** | نواری کوچک (مشابه ارتفاع taskbar) که نرخ‌ها را یکی‌یکی نمایش می‌دهد |
| **پنجره کامل** | با کلیک روی نوار باز می‌شود — همه نرخ‌ها به‌صورت یکجا + ماشین‌حساب تبدیل |

---

## ویژگی‌ها

### نوار اسلایدینگ
- نمایش متناوب نرخ ارزها با انیمیشن روان
- رنگ‌بندی سبز/قرمز بر اساس تغییر قیمت نسبت به روز قبل
- نمایش آخرین زمان بروزرسانی
- قابل جابجایی با drag به هر نقطه‌ای از صفحه
- قابل تغییر اندازه از دو طرف (resize)
- منوی راست‌کلیک: بروزرسانی دستی، تنظیمات، خروج

### پنجره کامل
- مشاهده همه نرخ‌ها به‌صورت یکجا
- **ماشین‌حساب تبدیل ارز**: تبدیل سریع تومان ↔ هر ارز خارجی
- پرچم‌های رنگی کنار نام هر ارز
- تازه‌سازی با دکمه ↺

### تنظیمات
- انتخاب سرویس API و وارد کردن کلید
- انتخاب ارزهای مورد نظر از لیست کامل
- تنظیم بازه بروزرسانی (۱۵ دقیقه تا ۲۴ ساعت)
- انتخاب حالت نمایش تغییر قیمت
- انتخاب نوع نمایش نماد ارز: فقط پرچم / فقط نماد / پرچم + نماد
- فعال/غیرفعال کردن اجرای خودکار با ویندوز
- ریست کامل تنظیمات

### آپدیت خودکار
- بررسی نسخه جدید در پس‌زمینه هنگام راه‌اندازی
- دانلود و جایگزینی خودکار exe با نمایش نوار پیشرفت
- نمایش راهنماهای کاربردی در حین دانلود

## راه‌اندازی اولیه

1. برنامه را اجرا کنید؛ نوار کوچک پایین-راست صفحه ظاهر می‌شود
2. روی نوار کلیک کنید تا پنجره کامل باز شود
3. دکمه **⚙ تنظیمات** را بزنید
4. یک سرویس API انتخاب و کلید API را وارد کنید:

| سرویس | ثبت‌نام رایگان |
|-------|----------------|
| نوسان | [navasan.tech](https://navasan.tech) |
| BRS API | [brsapi.ir](https://brsapi.ir) |
| آلان‌چند | [alanchand.com](https://alanchand.com/media/api) |

> **توجه:** API های رایگان محدودیت تعداد درخواست روزانه دارند.  
> بازه بروزرسانی را روی **۶ ساعت یا بیشتر** تنظیم کنید.

5. بازه بروزرسانی را تنظیم کنید (پیش‌فرض: ۶ ساعت)
6. **ذخیره** کنید

## ارزهای پشتیبانی‌شده

| دسته | ارزها |
|------|-------|
| ارز خارجی | دلار آمریکا، یورو، پوند، درهم امارات، لیر ترکیه، دلار کانادا، دلار استرالیا، فرانک سوئیس، ین ژاپن، یوان چین، روبل روسیه |
| رمزارز | بیت‌کوین (BTC)، اتریوم (ETH) |
| طلا و فلزات | طلا (گرم ۱۸)، طلا (اونس)، مثقال طلا، نقره |
| سکه | سکه بهار آزادی، نیم سکه، ربع سکه، سکه گرمی |

## حالت‌های نمایش

### نماد ارز
در تنظیمات بخش «نمایش» سه حالت قابل انتخاب است:

| حالت | نمونه |
|------|-------|
| فقط پرچم | پرچم رنگی کشور |
| فقط نماد | `USD` |
| پرچم + نماد | پرچم + نام ارز |

### تغییر قیمت
| حالت | توضیح |
|------|-------|
| رنگ فلش | فلش بالا/پایین رنگی |
| رنگ + مقدار تغییر | فلش + مقدار تغییر عددی |

مقدار تغییر نسبت به **قیمت پایانی روز قبل بازار** محاسبه می‌شود.

## استفاده

| عمل | نتیجه |
|-----|-------|
| کلیک روی نوار | باز/بسته کردن پنجره کامل |
| درگ روی نوار | جابجایی نوار |
| کشیدن لبه‌های نوار | تغییر عرض |
| راست‌کلیک روی نوار | منوی سریع (بروزرسانی، تنظیمات، خروج) |
| دکمه ▲ در پنجره کامل | باز کردن ماشین‌حساب تبدیل ارز |
| دکمه ↺ در پنجره کامل | بروزرسانی دستی |
| دکمه ⚙ در پنجره کامل | تنظیمات |

## داده‌ها و تنظیمات

همه فایل‌ها در `%APPDATA%\WinArz\` ذخیره می‌شوند:

| فایل | محتوا |
|------|-------|
| `config.json` | تنظیمات کاربر |
| `currency_tray.log` | لاگ‌های برنامه |
| `.tutorial_ticker` | وضعیت راهنمای نوار |
| `.tutorial_panel` | وضعیت راهنمای پنجره |

# WinArz — نرخ ارز لحظه‌ای برای ویندوز

نرم‌افزار رایگان نمایش لحظه‌ای نرخ ارز، طلا و سکه در نوار وظیفه ویندوز.

---

## دانلود

آخرین نسخه را از بخش [Releases](https://github.com/mojtabaniknam/WinArz-releases/releases) دانلود کنید.  
فایل `WinArz.exe` را اجرا کنید — نصب نیاز ندارد.

---

## ⚠ هشدار Windows SmartScreen

هنگام اولین اجرا ممکن است Windows پیام آبی‌رنگ **"Windows protected your PC"** نمایش دهد.  
این هشدار برای همه برنامه‌های بدون امضای دیجیتال طبیعی است و خطری ندارد.

**برای رد کردن:**

> ۱. روی **More info** کلیک کنید  
> ۲. دکمه **Run anyway** ظاهر می‌شود — روی آن کلیک کنید  
> ۳. برنامه اجرا می‌شود  

این هشدار فقط در اولین اجرا نمایش داده می‌شود.

---

## راه‌اندازی

۱. برنامه را اجرا کنید  
۲. روی نوار کلیک کنید تا پنجره کامل باز شود  
۳. دکمه **⚙ تنظیمات** را بزنید  
۴. سرویس API انتخاب کنید و کلید رایگان دریافت کنید:

| سرویس | ثبت‌نام |
|-------|---------|
| نوسان | [navasan.tech](https://navasan.tech) |
| BRS API | [brsapi.ir](https://brsapi.ir) |
| آلان‌چند | [alanchand.com](https://alanchand.com/media/api) |

> ⚠ API رایگان محدودیت روزانه دارد — بازه بروزرسانی را روی **۶ ساعت یا بیشتر** تنظیم کنید.
